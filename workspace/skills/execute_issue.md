# SKILL: EXECUTE_ISSUE

Responsável: Developer

1. Buscar issue atribuída:
   gh issue list --assignee @me --state open

2. Criar branch:
   feature/issue-{numero}

3. Implementar solução

4. Commit:
   feat: implementa #{numero}

5. Criar PR:
   gh pr create

6. Atualizar WORKING_DEV.md

Nunca trabalhar sem Issue.
Nunca commitar direto na main.
